Event-driven notifications. Listen for ANY event that uses the ti-common-event-handler module. An instance of this node will output a message from any ti event handler even name specified.

Inputs

There are no inputs. The node is triggered by an event produced by any code using the ti-common-event-handler.

Outputs

The node outputs a message for each matching event.

Node Settings

Name string
A short description shown in the admin interface
Event Name string

REQUIRED. The event name to match.
The defined string may include the character * (or +) to act as a single-level wildcard. Use ** or # for a multi-level wildcard. The / character acts as a level (namespace) separator for the wildcards.

Example

one/*/three will match events named one/two/three, one/99/three, etc.
one/** or one/# will match events named like one/two/three, or one/99.
